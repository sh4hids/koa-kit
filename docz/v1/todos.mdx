# Todos

## Create New Todo
This route is used to create a new todo task and it will return an object of newly created user.

#### Route (`post`)
```
https://koa-kit.com/api/v1/todos
```

#### Parameters (`body`)
| name      | Type   | Description                                    |
| --------- | ------ | ---------------------------------------------  |
| title | String | _Required_. The title of the task.        |
| description  | String | _Optional_. The description of the task.         |
| isDone     | Boolean | _Optional_. Current status of the task.             |

#### Response Example
```
{
  "success": true,
  "message": "New task added successfully!",
  "data": {
    "isDone": false,
    "_id": "5c11d51dfa01cb2807b01ca8",
    "title": "Test task 32",
    "description": "Lorem test description",
    "createdBy": "5c07a44e1ec68b10b771b7c5",
    "createdAt": "2018-12-13T03:42:21.366Z",
    "updatedAt": "2018-12-13T03:42:21.366Z",
    "__v": 0
  }
}
```

#### Route (`get`)
```
https://koa-kit.com/api/v1/users/:id/todos
```
